# comeon-in

`comeon-in` repo to manage login flows

## Pre-requisites

1. Install `nvm` to manage Node.js versions
   Follow instructions at: https://github.com/nvm-sh/nvm

2. Install `Node.js`
   ```bash
   nvm install
   ```

   This command will pick the required node version from `.nvmrc`.

3. Install `yarn`

   ```bash
   brew install yarn
   ```
